ViewStateActivity

interface ViewStateActivity : LifecycleOwner

Author

Davide Giuseppe Farella. An interface containing Lifecycle-driven observe extensions functions for Activity

Implements this interface on your Activity for be able to call LockedViewStateStore.observe and LockedViewStateStore.observeData without passing the LifecycleOwner explicitly. I.e. myViewStateStore.observeData( ::updateUi ) it's a valid function that will observe the data respecting the Activitys Lifecycle.
