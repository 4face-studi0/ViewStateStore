errorStateGenerator

var errorStateGenerator: ErrorStateGenerator

This is a generator for custom ViewState.Error

Set a new value for override the default behaviour and create a custom ViewState.Error for your Throwable. E.g. > ViewStateStoreConfig.errorStateGenerator = { throwable -> // this = ErrorStateFactory when( throwable ) { is ClassCastException -> MyClassCastViewStateError else -> /* this. */default } }
